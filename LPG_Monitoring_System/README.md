# Automated LPG Cylinder Monitoring & Safety System

 Overview
A smart IoT-enabled system that tracks LPG usage by weight and detects gas leakage to ensure household safety.

 Components
- Arduino UNO  
- MQ-2 Gas Sensor  
- Load Cell + HX711 Amplifier  
- 16x2 LCD Display  
- Buzzer + LED
